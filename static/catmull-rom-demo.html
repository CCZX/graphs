Catmull-Rom → 三次贝塞尔 转换演示

3个输入点 → 2段贝塞尔曲线

输入点 (P0-P2)
Catmull-Rom 曲线
返回值 c1（紫色）
返回值 c2（绿色）
返回值 to（金色）

核心概念：控制点是算出来的！

图一（二次贝塞尔）vs 我们的（三次贝塞尔）

图一的曲线是 二次贝塞尔：3个点（起点、控制点、终点），控制点需要人为指定

我们的曲线是 三次贝塞尔：4个点（起点、C1、C2、终点），C1/C2 由算法自动计算

返回值结构

函数返回 BezierSegment[]，每个元素包含三个点：

{ c1: Point, c2: Point, to: Point }

• c1（紫色方块）：第一个控制点

• c2（绿色方块）：第二个控制点

• to（金色圆点）：曲线终点

起点隐含为上一段的 to（或第一个输入点）

曲线为什么经过 P0 和 P1？

三次贝塞尔曲线公式：

B(t) = (1-t)³×P0 + 3(1-t)²t×C1 + 3(1-t)t²×C2 + t³×P1

当 t=0（起点）：B(0) = P0，曲线经过起点

当 t=1（终点）：B(1) = P1，曲线经过终点

**控制点 C1/C2 只影响曲线形状，不影响端点！**

控制点怎么算的？

公式：C1 = P1 + (P2 - P0) / 6，C2 = P2 - (P3 - P1) / 6

首段特殊处理：P0=P1，末段特殊处理：P3=P2
